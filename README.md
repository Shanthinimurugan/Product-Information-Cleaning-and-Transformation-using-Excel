# Product-Information-Cleaning-and-Transformation-using-Excel

 This project is a process of detecting and correcting data into restructuring a product information from a data analysis perspective.

## Description

This project is an Excel-based Product information cleaning and transformation assignment. It focuses on cleaning and transformation in various product details such as Product ID, Product Name, Brand Name, Price, Quantity, and Category. In this project, we used data cleaning techniques, formatting tools, and transformation operations to perform an analysis.

Product information cleaning focuses on identifying and correcting issue in raw product data. And Product information transforming focuses a cleaned data into standardized structure and format suitable for business needs.

The primary aim of this project refers to the process of improving, standardizing and preparing product data so it is accurate, consistent and usable across business systems.


# Getting Started

## Dependencies

The following requirements are needed:

* Operating System: Windows 10 or later
* Software: Microsoft Excel 2016 or later
* Prior knowledge of cleaning and transformation functions in Excel


# Installing

* Download the attached Excel files from this repository.
* Save the files to your computer.
* Open the file using Microsoft Excel. 
* If Excel displays a Protected View Warning, click "Enable Editing".


# Executing Program

## Steps to Run the Project

1. Open the attached Excel file: ```Product Information Cleaning and Transformation.Xlsx``` and ```Raw Data.xlsx```
2. View the Excel workbook.
3. Check the output cell highlighted green colour in the file to explore the data and compare the Raw data file to review the newly added columns.


# Help

## Common Issues

* Check whether Microsoft Excel is installed properly.
* Incorrect cell selection may lead to misinterpretation of anaysis and examination.
